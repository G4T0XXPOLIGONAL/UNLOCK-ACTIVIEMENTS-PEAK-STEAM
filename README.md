# ⛰️ PEAK Ultimate Unlocker - by G4T0XX

Mod simples e direto para desbloquear 100% das Badges (Conquistas) do jogo PEAK instantaneamente!

## 🚀 Como Instalar e Usar
1. Instale o [MelonLoader](https://melonwiki.xyz/#/) (v0.6.0+) na pasta principal do seu jogo PEAK.
2. Abra o jogo uma vez até o menu e feche (para gerar as pastas necessárias).
3. Baixe o arquivo `.dll` na aba **Releases** deste GitHub.
4. Coloque a `.dll` dentro da pasta `Mods` do jogo.
5. Inicie o jogo. No menu principal, aperte a tecla **INSERT**, clique no botão para desbloquear tudo e reinicie o jogo para sincronizar com a Steam!
